# Kiki Chatbot

I made a chatbot using Python. The Kiki chatbot can understand things you say to it like greetings or when you ask for the date and time. It can also do addition tell jokes and have a conversation with you.

I wanted to practice using Python so I built this project. I learned about functions, conditionals, loops and how to handle exceptions. I also learned about modules and how to get input from users.

---

## Features

- The Kiki chatbot can say hello to you

- It can give you responses

- It can show you the current date

- It can show you the current time

- It can do basic addition for you

- It can tell you jokes

- You can exit the chatbot easily

- If you say something the Kiki chatbot does not understand it will still be nice to you

---

## Built With

- I used Python 3 to build the Kiki chatbot

- I used the `module to make the chatbot give random responses

- I used the `datetime` module to get the current date and time

## Getting Started

### Prerequisites

- You need to have Python 3 installed on your computer

### Run the project

First you need to get the project from the internet:

```bash

git clone https://github.com/your-username/kiki-chatbot.git

```

Then you need to go to the project folder:

```bash

cd kiki-chatbot

```

Now you can run the Kiki chatbot:

```bash

python chatbot.py

```

## Commands

| Command | Description |

|----------|-------------|

| hello / hi / hey | The Kiki chatbot will say hello to you |

name | The Kiki chatbot will tell you its name |

| how are you | The Kiki chatbot will give you a random response |

| time | The Kiki chatbot will show you the time |

| date | The Kiki chatbot will show you todays date |

| add 5 10 | The Kiki chatbot will add two numbers for you |

| joke | The Kiki chatbot will tell you a joke |

| bye | You can exit the Kiki chatbot |
